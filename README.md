System Requirement:

- Phonegap V3.0
http://phonegap.com/

- IOS Simulator
You can use xCode Simulator or run the command brew install ios-sim to install it and then run the game

Info about ios-sim 
https://github.com/phonegap/ios-sim

Info about brew
http://brew.sh/

- To run the game 

Inside the game folder run the next command

phonegap run ios
